# One Level Community Bridge Demo

This folder contains a third website concept for One Level Energy NFP.

Design direction:

- More civic and community-centered than the AI-led v2 concept
- Strong pathway storytelling with a cleaner, softer visual tone
- Built to demo well on Netlify and still transfer cleanly into Wix

Key files:

- `index.html`: homepage
- `about.html`: leadership and mission page
- `styles.css`: shared styling
- `script.js`: form behavior and demo-fill flow
- `site-config.js`: lead endpoint and demo config

This concept uses the same lead capture structure as the v2 demo so both demos can feed the same Google Sheets pipeline.
